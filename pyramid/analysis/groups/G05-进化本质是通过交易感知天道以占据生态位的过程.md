# G05: 进化本质是通过交易（T）感知天道（N）、在法则（R）内外探索以占据生态位（NI）的过程

> 进化并非随机突变，而是主体通过交易（T）感知天道（N）变化，在法则（R）内外探索，占据生态位（NI）并在必要时突破法则寻求与新天道相容的过程。

## 组内逻辑顺序

本组 atoms 按“理论定义 -> 阶段演变 -> 族群分化 -> 实践策略”的逻辑排列。首先定义进化的核心公式 f(N, R, T) 及关键概念（N/R/T/NI）；其次阐述进化的三/四阶段模型及当前的“分化时刻”背景；接着分析不同族群（麻瓜/炒家/Cyber-Taoist）在剧变期的命运分化；最后落实到具体的行动策略（OpenClaw 实验、KL 系列项目）及资格积累路径。
