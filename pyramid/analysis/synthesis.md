# 收敛（Synthesis）

从 [groups/](groups/) 的观点句中归纳出的顶层观点候选。这些候选是整个知识体系最高层的洞察，也是 structure/ 视角展开的起点。

## 顶层观点候选

| 编号 | 候选观点 | 支撑 Groups | 状态 |
| ---- | -------- | ----------- | ---- |
| S1 | 进化本质是主体通过交易（T）感知天道（N）变化，在法则（R）内外探索以占据生态位（NI）的过程 | G01, G04, G05 | 正式 |
| S2 | 个体在 AI 时代的生存法则在于"知 R/N、行 T、占 NI、顺势"四大原则 | G02, G04, G05 | 正式 |

### 待成熟候选

| 编号 | 候选观点 | 支撑 Groups | 状态 |
| ---- | -------- | ----------- | ---- |
| S3* | 人类与 AI 的协作深度将导致物种分化，唯有信息处理能力超越 AI 者方能成为新贵族 | G05 | 待成熟 |

## 候选间的关系

- **S1 是底层公理**：定义了进化的通用公式 $E = f(N, R, T)$，其中 N（天道）为自变量，R（法则）为防火墙，T（交易）为感知探头，NI（生态位）为法则内定位。为 S2 和 S3* 提供了理论基石。G05 的加入进一步验证了该公式在 Cyber-Taoist 理论中的核心地位。
- **S2 是 S1 在 AI 时代的具体行动指南**：将 S1 中的抽象概念映射为四大原则——知 R/N（理解法则、感知天道）、行 T（通过交易探测天道）、占 NI（占据生态位但不执着）、顺势（判断何时在法则内优化、何时突破法则感应天道）。G05 中关于"资格积累"和"共识创造"的原子强化了这一策略的可操作性。"控 R"（如本地部署等手段）是"知 R/N"原则下的具体战术，属于 S2 策略的实施细节，而非独立的顶层观点。
- **S3\* 是进化的终极演化结果与筛选标准**（待成熟）：基于 S1-S2 的实践，指出进化的终局是物种分化。只有那些通过 S2 策略最终实现信息处理能力超越 AI 的个体，才能在新生态中占据"新贵族"生态位（NI）。这是 S1 公式在长周期下的必然推论。
- **G05 作为全量理论组**：整合了 ER、CT、CC 系列原子，全面支撑了 S1（进化公式）、S2（四大原则）以及待成熟的 S3*（物种分化与终极目标）。
