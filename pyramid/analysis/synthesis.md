# 收敛（Synthesis）

从 [groups/](groups/) 的观点句中归纳出的顶层观点候选。这些候选是整个知识体系最高层的洞察，也是 structure/ 视角展开的起点。

## 顶层观点候选

| 编号 | 候选观点 | 支撑 Groups | 状态 |
| ---- | -------- | ----------- | ---- |
| S1 | 进化本质是环境规则（R）约束下的价值交易（T）与生态位（N）占据过程 | G01, G04 | 正式 |
| S2 | 个体在 AI 时代的生存法则在于“知规则、创共识、占需求”的三位一体策略 | G02, G04 | 正式 |
| S3 | 掌控 AI 边界并实施本地化部署是成为“新贵族”的核心战略 | G03 | 正式 |

## 候选间的关系

- **S1 是底层公理**：定义了进化的通用公式 $E = f(R, T, N)$，为 S2 和 S3 提供了理论基石。
- **S2 是 S1 在 AI 时代的具体行动指南**：将 S1 中的抽象概念映射为具体策略（知 R、创 T/共识、占 N/需求）。
- **S3 是 S2 的高阶战略目标与防御升级**：在 S2“知 R"的基础上，进一步要求“控 R"（通过本地部署掌控 AI 边界），旨在应对物种分化风险，实现从普通生存到“新贵族”的阶层跃迁及安全生态位保护。
- **G04 作为实证组**：同时支撑了 S1（验证 R/T/N 模型）和 S2（验证具体生存策略的有效性）。
