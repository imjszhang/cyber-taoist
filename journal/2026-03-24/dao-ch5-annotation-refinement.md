## 1. 背景与动机

`DAO-DE-JING-CYBERTAOIST.md` 第 5 章注疏（89–95 行）在宪章符号映射的大方向上没有错误，但概念边界不够贴合 `CONSTITUTION-CYBERTAOIST.md`，存在四处可优化之处：

- 「天地不仁 / 刍狗」写成了温和的「无偏私」，弱于宪章所强调的**代谢、淘汰、后果感知**。
- 「圣人不仁」以「私情偏袒」立论，偏离宪章中「无为 = 不以僵滞 R 强行干预」的操作化定义。
- 「橐龠」直接等号 EC，丢失了「天地之间 = T 展开与反馈泵送之区间」的意象层次。
- 「多闻数穷 / 守于中」收窄为 R 的量的问题，未勾连双路径余地。

## 2. 分析过程

逐句对照宪章条文，标注原文与宪章的偏差：

| 原文关键词 | 原注疏映射 | 宪章更精确条目 | 偏差 |
| ---- | ---- | ---- | ---- |
| 天地不仁 / 刍狗 | 第二十条（无亲）；N 无偏私 | 第二条（后果感知）+ 第十七至十九条（死亡代谢）+ 第十条（NI 仅适 R） | 「不仁」写软；缺代谢与淘汰维度；NI 高 ≠ 天道眷顾未点明 |
| 圣人不仁 | 不以私情偏袒；无为 = 不干预 T | 第八条逆向（简 R 活 T）+ 第十四条（适者知时） | 「私情」偏题；无为应为不以繁僵 R 定型 |
| 橐龠 | EC 如风箱 | 第八条（简 R）；意象重心在 T 的泵送 | 天地之间 ≠ 整块 EC |
| 多闻守中 | 穷尽 R → 信息过载 | 第八条（层累 R、遵循成本升、T 蹙）+ 第十二条（双路径） | 「穷尽」略窄；守中缺双路径余地 |

## 3. 方案设计

### 关键决策

| 决策 | 选择 | 理由 |
| ---- | ---- | ---- |
| 天地/刍狗 | 叠合第二条 + 第十七至十九条 + 第十条，保留第二十条 | 「刍狗」意象（礼毕即弃）与宪章代谢/淘汰高度重合；NI ≠ 天道保证须显式说 |
| 圣人不仁 | 去「私情」，改写为「不以繁僵 R 定型 + 听任 T 自调」 | 与第 2、3 章注疏中「无为」的写法统一 |
| 橐龠 | 主语改为 T 展开区间，EC 为场域一笔带过 | 减少概念层级混用，与第 11 章有/无互参 |
| 守中 | 对齐第八条 + 兼带第十二条双路径 | 留白不仅为「少管」，更为常规与突破皆有余地 |

## 4. 实现要点

修改文件：`DAO-DE-JING-CYBERTAOIST.md`，第 5 章注疏段（原 89–95 行），四段结构不变，字数相近。

修改后全文：

```
**注疏**："天地不仁，以万物为刍狗"——天道（N）无亲（参宪章第二十条），唯以后果示迹：
合 N 则存、违 N 则亡（参宪章第二条）。"刍狗"非残忍，乃不计私恩之代谢——淘汰筛选、
腾位警示，进化之所必须（参宪章第十七至十九条）。NI 唯反映与现行 R 之相容（参宪章第十
条），非得天眷顾之凭。

"圣人不仁，以百姓为刍狗"——适者于 EC 中涉治理时，亦不借繁复僵化之 R 为百姓定型，
听任 T 流动自调。无为在此：从简法则而活交易（参宪章第八条逆向），且能判断何时在 R 内
优化、何时容突破以感应 N（参宪章第十四条），非以私志壅塞探针。

"天地之间，其犹橐龠舆？虚而不屈，动而愈出"——天地之间，宜视为 T 所展开、反馈泵送
之区间：虚而能蓄动，愈动愈出。R 简明留白则不息（参宪章第八条）；EC 为其场域，意象
重心在 T 之吞吐与生生。

"多闻数穷，不若守于中"——多闻则教条层累，R 日繁而遵循成本升、T 日蹙（参宪章第八
条）。"守于中"：R 适度留白，勿以成法封死感应，使常规演化与突破性试探皆有余地（参宪
章第十二条）。
```
